# 🧱 Web Tetris

**[Play Web-Tetris](https://KiloFarsh.github.io/Web-Tetris/)**

### PC Version:
* **Arrows (← →):** Move left/right.
* **Arrow Up (↑):** Rotate piece.
* **Arrow Down (↓):** Fast fall.
* **Space:** Pause/Resume.
* **H:** Hard Drop.
* **R:** Restart game.

### Mobile Version:
* Use the **on-screen touch buttons** to move, rotate, and drop pieces.
